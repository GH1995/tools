如何使用文档
-----------------------

帮助的题头
-----------

文件的第一行应该包括帮助文件的文件名，接下来是一行对插件功能的描述。

在帮助文件中用星号括起一个词创建了一个可以用于跳转的"tag"。

写什么文档
----------------

* Introduction
* Usage
* Mappings
* Configuration
* License
* Bugs
* Contributing
* Changelog
* Credits

内容目录
-----------------

有很多事情需要在这里提一下。
首先，`=`字符组成的那行将被高亮。你可以用这些行醒目地隔开你的帮助文档各部分。
你也可以使用`-`隔开子段落，如果想要的话。

Vim将隐藏`*`和`|`并高亮其间的内容，所以用户将会看到一个整洁漂亮的目录，以便于跳到感兴趣的地方。
